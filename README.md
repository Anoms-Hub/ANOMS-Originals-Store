## 🏪 What Is ANOMS-Originals-Store?

ANOMS-Originals-Store is the **e-commerce backbone** for Anom Originals. It manages product definitions, mockups, store listings, pricing strategies, third-party integrations, and automation workflows that power the AO storefront.

This repo bridges the creative side (art + brand) with the technical side (app + ops) to bring AO products to market.

---

## 📂 Folder Structure

```
ANOMS-Originals-Store/
├── products/        # 📦 Product definitions and specifications
├── mockups/         # 🖼️ Product mockups and visual previews
├── listings/        # 📝 Store listing copy, descriptions, and SEO
├── pricing/         # 💰 Pricing models, tiers, and discount rules
├── integrations/    # 🔌 Third-party platform connections
├── automation/      # ⚙️ Automated workflows for store operations
└── README.md
```

---

## 📋 What Goes Where

| Folder | Purpose | Examples |
|--------|---------|---------|
| **products/** | Product specs and definitions | Item details, variants, SKUs, digital asset packages |
| **mockups/** | Visual product previews | T-shirt mockups, sticker sheets, digital product previews |
| **listings/** | Store-facing content | Product titles, descriptions, tags, SEO keywords |
| **pricing/** | Revenue and pricing strategy | Price tiers, bundle discounts, promotional pricing |
| **integrations/** | Platform connectors | Shopify configs, payment gateway setup, fulfillment APIs |
| **automation/** | Operational workflows | Inventory alerts, order processing, listing sync scripts |

## 🔄 Product Pipeline

```
🎨 Asset created in anom-artsy
    ↓
🏷️ Brand approved via ANOMS-Brand-Kit
    ↓
📦 Product defined → products/
    ↓
🖼️ Mockup created → mockups/
    ↓
📝 Listing written → listings/
    ↓
💰 Price set → pricing/
    ↓
🚀 Published to storefront via integrations/ + automation/
```
